### Project 4 - Rock, Paper, Scissors
***
## Instructions
Make a rock, paper, scissors game.

Start the game by asking the player:

```python
"What do you choose? Type 0 for Rock, 1 for Paper or 2 for Scissors."
```

## Example Input
```python
"What do you choose? Type 0 for Rock, 1 for Paper or 2 for Scissors."
0
```
